Into The Mind

Jessie Estrada,George Alvarado, and I build "Jump into the mind" in a two Day Hackaton with SOMOS in collaboration of Activition and Blizzard. There will be obstacles in the way to be careful! This game takes place inside someone's mind and the chest symbolizes the goals and aspirations we all have. While we make an attempt to achieve these goals, there are situations we do not expect. Along the wya, there are also tools we can use to help us on our journey. We created the different "power-up" effects. (The orange, cake, and chocolate bar) Our teammate George also created other power ups that we did not have time to implement. These were: a donut, a taco, and kebab. We did not create music for this game. We wanted to create our own sound track for the gameplay, but we were not able to finish anything complete on time. Github-Repo: https://github.com/bryanmax9/ignite_game_jam

Guardians of the Hatchlings

The "Guardians of the Hatchlings" website showcases a UI/UX design for interfacing with my machine learning model hosted on AWS Lambda, developed using Amazon SageMaker and S3 for model training and data storage, respectively. This front-end was designed to manage POST requests where users could upload images, and the API would identify whether the subject was a turtle. Although the cloud-based API has since been decommissioned due to budget constraints, the comprehensive documentation on my Github Repository highlights my technical proficiency and thorough approach to project documentation.Visite Website🌐
